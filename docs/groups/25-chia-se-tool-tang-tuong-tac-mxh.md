## Nội quy group

```
📋 NỘI QUY — CHIA SẺ TOOL TĂNG TƯƠNG TÁC MXH

✅ ĐƯỢC PHÉP:
1. Review tool tăng tương tác cho bất kỳ MXH nào (kèm proof)
2. Hỏi đáp về cách tăng tương tác hiệu quả cho từng nền tảng
3. Chia sẻ kinh nghiệm xây dựng tài khoản MXH từ đầu
4. So sánh, phân tích các tool theo tiêu chí khách quan
5. Cảnh báo tool lừa đảo, không hoạt động (có bằng chứng)
6. Chia sẻ update mới nhất về thuật toán các MXH

❌ CẤM:
1. Quảng bá tool vi phạm Terms of Service của các MXH
2. Spam link không liên quan đến chủ đề tăng tương tác
3. Review giả, tự khen tool của mình
4. Chia sẻ tool hack tài khoản, đánh cắp dữ liệu
5. Bán tài khoản MXH các loại
6. Tạo hoặc chia sẻ bot comment rác, spam

📌 FORMAT KHI ĐĂNG REVIEW TOOL:
- Tên tool + link chính thức
- Nền tảng hỗ trợ
- Loại tương tác: like/follow/view/comment
- Giá (miễn phí / trả phí)
- Proof thực tế (ảnh chụp màn hình)
- Điểm đánh giá: ⭐/10

⚠️ Vi phạm → Xóa bài → Cảnh cáo → Ban
```
