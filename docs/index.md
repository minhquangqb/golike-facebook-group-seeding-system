# GoLike Facebook Groups System

Hệ thống quản lý và phát triển Facebook Groups cho GoLike.

## Mục tiêu

- 30 group hoạt động (có tương tác thật)
- 10.000+ member thật sau 30 ngày
- Kéo user làm việc cho GoLike
- Kéo khách hàng mua các jobs seeding
